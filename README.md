# \# 🔐 Password Manager (Python)

# 

# A simple command-line Password Manager built with Python.

# 

# It allows you to securely store, edit, search, and manage your passwords using a JSON file for storage. The application also includes a random password generator and PIN-protected password reveal.

# 

# \## Features

# 

# \- View saved passwords (passwords are hidden by default)

# \- Add new passwords

# \- Edit existing passwords

# \- Delete passwords

# \- Search websites

# \- Reveal passwords using a master PIN

# \- Generate secure random passwords

# \- Automatic alphabetical sorting of entries

# \- Passwords stored locally in a JSON file

# 

# \## Technologies Used

# 

# \- Python 3

# \- JSON

# \- os

# \- random

# \- string
